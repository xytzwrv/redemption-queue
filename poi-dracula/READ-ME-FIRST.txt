RANDOM POI — DRACULA EDITION
============================

A Dracula-themed fork of the POI selector (../poi). Same feature set —
pick an Apex map, spin, land on a random POI — but restyled as a modern,
rounded, smooth (anti-aliased) UI in the Dracula palette instead of the
1-bit pixel look.

  - Rounded cards, soft shadows, Inter font, smooth region wireframes.
  - The theme selector in the FX console (press H) now picks DRACULA
    PATTERNS: Purple, Pink, Cyan, Green, Orange, Red, Night, Alucard.
  - The selected/landed region gets a soft translucent glow (not a
    dithered one).

  READ-ME-FIRST.txt  - this file
