Slinkystream — Overlay Dashboard

Every OBS element in this repo, in one place. "Open" launches the standalone page; "Copy URL" grabs the exact link to paste into an OBS Browser Source, resolved against wherever this page is being served from. Tip: load this dashboard itself via http://127.0.0.1:8777/dashboard.html rather than localhost - every copied link then works everywhere, including Now Playing, whose Spotify login specifically requires 127.0.0.1.
